# My Portfolio

Um portfólio pessoal de front-end com foco em design moderno e efeitos visuais sutis.

## Sobre

Este projeto é uma landing page estática criada para apresentar um perfil de engenheiro full-stack com foco em segurança e sistemas de alto desempenho. A página inclui seções de trabalho, tecnologias utilizadas e contatos.

## Tecnologias

- HTML
- CSS
- JavaScript
- Fontes do Google Fonts: Plus Jakarta Sans e IBM Plex Mono

## Estrutura do projeto

- `index.html` — marcação da página
- `style.css` — estilos visuais, layout responsivo e animações
- `script.js` — efeitos de spotlight, transição de fade-in e revelação ao rolar a página

## Recursos

- Navegação fixa com links para seções internas
- Hero section com título, badge e descrição
- Grid de projetos em destaque
- Seção de stack/core architecture
- Footer com links sociais
- Efeitos de spotlight e noise de fundo
- Animações suaves de entrada e revelação ao scroll

## Como usar

1. Abra `index.html` no seu navegador.
2. Ou execute um servidor local se preferir (por exemplo, usando `Live Server` no VS Code).
